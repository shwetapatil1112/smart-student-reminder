# Smart Student Reminder System

A simple Python desktop app to help students manage their tasks and assignments by setting reminders.

---

## Features

- Add reminders with **Title**, **Date (YYYY-MM-DD)**, and **Time (HH:MM)**  
- Stores reminders in a local SQLite database  
- Shows confirmation when a reminder is saved  
- Simple and user-friendly interface with Tkinter

---

## Technologies

- Python 3  
- Tkinter (GUI)  
- SQLite (Database)

---

1. Install Python 3 if not already installed.  
2. Clone or download this repository.
